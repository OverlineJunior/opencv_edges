Programa desenvolvido em _C++_ com a biblioteca _OpenCV_ para processar uma imagem em tons de cinza, aplicar o filtro Laplaciano e realçar as bordas da imagem original.

## Compilação

1. Instalar o [_xmake_](https://xmake.io/guide/quick-start.html)
2. Abrir um terminal neste diretório
3. Usar `xmake` para compilar
4. Usar `xmake run` para executar

## Dependências

1. OpenCV 4.12.*

As dependências são automaticamente geridas pelo *xmake*.

## Ajuda

Caso não seja possível usar o _xmake_, há um executável compilado no _Windows_ em `opencv_edges/build/windows/x64/release/opencv_edges.exe`.
